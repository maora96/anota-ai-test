# anota-ai-back-end-test

repository for the anota ai back-end dev test

## technologies

- nodejs
- mongodb
- express

## dependencies

- express
- body-parser
- cors
- dotenv
- mongoose

## start project

```bash
# clone repository
$ git clone https://github.com/maora96/anota-ai-test.git

# install Dependencies
$ npm install

# run app
$ npm start
```
